# To-Do-List
The To-Do List Application is a simple and user-friendly task management system that allows users to add, view, edit, mark, and delete tasks. It helps users organize their daily activities efficiently and improve productivity.
This project demonstrates basic concepts of file handling, data structures, and user interaction in a command-line environment.
